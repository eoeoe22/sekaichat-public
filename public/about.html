Google Gemini 기반 다중 캐릭터 챗봇 프로젝트
프로젝트 개요

세카이 채팅은 다양한 캐릭터와 대화할 수 있는 다중 캐릭터 AI 챗봇 서비스입니다. 실시간 대화, 이미지 업로드, 캐릭터 프롬프트 및 커스텀 이모지를 지원하며, Google Gemini API 기반의 자연스러운 언어 처리와 주제별 작업 모드를 제공합니다.

이 사이트는 비영리 2차 창작물입니다.

서버비 0원 / 수익 0원

주요 특징
  • 여러 캐릭터와의 멀티 대화, 초대/호출/챗봇들간 자동 호출
  • 개인 Gemini API 키 등록/사용자 닉네임/자기소개 설정
  • 자연스러운 대화의 기본 모드, 유용한 답변을 제공하는 작업 모드 지원
  • 실시간 이미지 업로드 및 분석
  • 커스텀 이모지 및 상황 설정 기능

개인정보/운영 안내
  • 서버·DB는 Cloudflare Workers/D1에 보관되며, 비밀번호는 해시+솔팅 암호화하여 저장합니다.
  • 개인 Gemini API키는 사용자가 직접 입력시 서버 DB에 저장하고, 삭제 버튼 클릭시 즉시 완전 삭제됩니다.
  • 모든 대화 데이터/파일은 Gemini API 이외 외부 사이트로 전송하거나 이용하지 않습니다
AI 모델 및 기술 스택

Google Gemini 가 챗봇의 핵심을 담당하며, 대화와 이미지를 처리합니다.
