프로젝트 개요

세카이 채팅은 다양한 캐릭터와 대화할 수 있는 다중 캐릭터 AI 챗봇 서비스입니다. 실시간 대화, 이미지 업로드, 캐릭터 프롬프트 및 커스텀 이모지를 지원하며, Google Gemini API 기반의 자연스러운 언어 처리와 주제별 작업 모드를 제공합니다.

이 사이트는 비영리 2차 창작물입니다.

서버비 0원 / 수익 0원

주요 기능

다중 캐릭터 대화

여러 AI 캐릭터를 하나의 대화에 초대하여 그룹으로 대화할 수 있습니다.

AI 미연시

AI 캐릭터와 교감하며 애정도를 쌓아가는 미연시 모드를 즐겨보세요. 당신의 선택이 캐릭터와의 관계를 변화시킵니다. (개발 중)

나만의 캐릭터 제작

자신만의 AI 캐릭터를 직접 만들고 대화할 수 있습니다. 캐릭터의 성격, 배경, 말투를 직접 설정해 대화에 초대할수 있습니다.

이미지 생성 & 업로드

Gemini의 멀티모달 기능으로 이미지를 인식하며, AI가 직접 그림을 그려주는 이미지 생성 기능도 지원합니다.

자유로운 커스텀

닉네임, 자기소개등 다양한 설정을 통해 자신만의 환경을 만들 수 있습니다.

두가지 대화 모드

자연스러운 대화를 위한 '기본 모드'와, 유용한 답변을 제공하는 '작업 모드'를 지원합니다.

이벤트 스토리 RAG 검색

프로젝트 세카이 이벤트 스토리 전체를 임베딩한 벡터 데이터베이스에서 연관성 기반 RAG검색이 가능합니다.

TTS 음성 합성

프로젝트 세카이 캐릭터들의 목소리로 채팅 메시지를 음성으로 들을 수 있습니다. 채팅 메시지의 버튼을 클릭하면 바로 재생해볼 수 있습니다.

모델 : ProsekaTTS
Gemini Native 이미지 생성

AI 에나낭이 직접 그려주는 이미지 생성 기능을 체험해보세요.
간단한 요청만으로 다양한 스타일의 그림을 생성할 수 있습니다.
현재 Gemini 2.5 Flash 이미지 생성 기능이 불안정해 2.0 Flash 모델로 구동중)

개인정보/운영 안내

  • 모든 대화 데이터/파일은 Gemini API 이외 외부 사이트로 전송하거나 이용하지 않습니다
